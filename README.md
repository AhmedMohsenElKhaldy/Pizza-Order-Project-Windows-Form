# 🍕 Pizza Order System (Windows Forms)

A simple Pizza Ordering Application built using C# and Windows Forms.

## Features
- Choose pizza size
- Select toppings
- Choose crust type
- Dine-in or Take-away
- Calculate total price automatically

## Technologies
- C#
- Windows Forms
- .NET Framework
